ofw
===

Open Fantasy World


Requirements:
=============

* CMake, http://www.cmake.org/cmake/resources/software.html
* Irrlicht 1.8, http://downloads.sourceforge.net/irrlicht/irrlicht-1.8.zip


Building:
=========

Linux:
------

> git clone https://github.com/Virako/ofw.git
> cd ofw
> mkdir build
> cd build
> cmake ..
> make
> ./ofw
